🔑 Token do GitHub

O painel salva os dados diretamente no seu repositório via GitHub API. Para isso, você precisa de um Personal Access Token. O token fica salvo só no seu navegador — nunca no código.

1
Acesse github.com → Settings → Developer settings → Personal access tokens → Tokens (classic)
2
Clique em Generate new token (classic), dê um nome e marque o escopo repo
3
Copie o token gerado e cole no campo abaixo

🔒 O token é salvo apenas no localStorage do seu navegador. Cada computador precisa configurar uma vez.

📁 Como funciona

Ao salvar um curso, o painel cria ou atualiza o arquivo data.json no repositório yasvizeu/decole_interno. O site vitrine lê esse mesmo arquivo para exibir os cursos ao público.

Repositório: yasvizeu/decole_interno Arquivo: data.json Formato: { "cursos": [ ... ], "updatedAt": "..." }

⚠️ O repositório precisa ser público para o site vitrine conseguir ler os dados.
